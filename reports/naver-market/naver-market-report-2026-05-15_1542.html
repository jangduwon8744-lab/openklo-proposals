# 네이버 농산물 시장조사 보고서

## 1. 오늘의 시장 요약
- 데이터 기반
  - 상위 노출 상품의 핵심 키워드는 ‘고당도’(22회), ‘씨없는’(20회)가 지배적이다. 품종명은 블랙사파이어, 크림슨, 애플청포도, 어텀/오톰크리스피, 세이블이 반복 노출된다. 20브릭스(20brix) 표기가 프리미엄 근거로 활발히 쓰인다.
  - 상품명 구성은 ‘키워드(고당도/씨없는) + 품종명 나열 + 용량’ 형태가 주류이며, 모음전/혼합옵션이 많다. 대용량(2~6kg) 옵션도 상위권에 다수.
  - 실측 가격·리뷰 데이터는 부족하나, 일부 노출에서 1~2kg 26,000원, 프리미엄 신품종 32,000원 단가가 관찰된다.
  - 산지직송 강조는 제한적(1건)이며, 다수는 수입·혼합 구성.
- 전략적 추정
  - 네이버에서 즉시전환보다 검색-뷰/블로그-리뷰-스토어 신뢰가 누적되며 전환을 만든다. ‘고당도/씨없는/브릭스’는 클릭 유도어, ‘산지 스토리/리뷰 사진/보관법’은 구매결정어.
  - 품종명 다발(모음전)은 다양한 롱테일 검색(예: “블랙사파이어 2kg”, “어텀크리스피 20브릭스”)을 동시에 흡수하려는 전략으로 보인다.
  - 프리미엄 포지션은 ‘브릭스 수치 + 항공/직송 + 신품종 스토리’ 조합이 먹힌다.

## 2. 네이버 검색 노출 상품 특징
- 데이터 기반
  - 제목에 품종명 다중 삽입(블랙사파이어/크림슨/애플청포도/오톰크리스피/세이블/거봉 등) + ‘고당도/씨없는’ 반복.
  - 대용량형(2~6kg), 가성비형이 주류, 프리미엄형은 상대적으로 적지만 ‘20브릭스/항공직송’ 등 명확한 근거 제시.
  - KF365·대형 유통 이미지도 노출(500g 팩) → 신뢰 대리 효과.
- 전략적 추정
  - 썸네일은 과실 클로즈업, 물방울/광택, 20brix/씨없는 배지형 문구로 CTR을 끌어올리는 전형을 사용.
  - 네이버는 제목 키워드 적합성 + 썸네일 CTR + 리뷰 신뢰 + 스토어 지수의 합성 신호로 랭킹을 조정하는 경향.

## 3. 반복 키워드 분석
- 데이터 기반(빈도)
  - 고당도: 22회, 씨없는: 20회, 프리미엄: 4회, 제철: 2회, 20브릭스/20brix: 4회.
  - 품종 다빈도: 블랙사파이어(가장 빈번), 크림슨, 애플청포도, 어텀/오톰크리스피, 블랙 세이블, 아도라, 샤인머스켓(소수), 레드글로브(소수), 델라웨어(소수).
- 전략적 시사점
  - 제목 구조 템플릿(추천): [씨없는/고당도 + 품종 1핵심] + [용량/과중] + [브릭스/산지 신호] + [배송/신선/보관 키워드]
    - 예시: 씨없는 고당도 블랙사파이어 2kg | 18~21brix | 당일선별 냉장발송

## 4. 가격/구성/옵션 흐름
- 데이터 기반
  - 관찰 가격: 1~2kg 26,000원(가성비형), 프리미엄 신품종 32,000원(소포장 추정). 500g 팩(대형 유통) 다수. 2~6kg 대용량/혼합 옵션 빈번.
  - 비정상가(1,990,000원) 존재 → 옵션/테스트성 노출로 보임.
- 전략적 추정(시세 범위, 지역·시즌 변동 큼)
  - 수입 씨없는 1kg: 13,000~20,000원, 2kg: 24,000~36,000원
  - 프리미엄(20brix/항공): 1kg 18,000~28,000원, 2kg 32,000~45,000원
  - 대용량 B2B(카페/주스용/못난이): 3~5kg 3.5만~7만대
  - 옵션 전략: 500g 체험 → 1kg 메인 → 2kg 패밀리 → 3~4kg 카페 → 혼합 시식팩(신품종 포함)으로 단계별 체리피킹 유도.

## 5. 스마트스토어형 상세페이지 전략
- 데이터 기반 인사이트 반영
  - 1스크린: 씨없는/고당도/브릭스 수치/주력 품종 1개만 전면 배지화. 과한 키워드 나열 지양, 핵심 가치 명료화.
  - 2스크린: 산지·선별·냉장체인 스토리(사진 3장: 수확/브릭스계 측정/선별기준).
  - 3스크린: 옵션 그리드(500g/1kg/2kg/3~4kg), 용도 매칭(간식/가족/카페).
  - 4스크린: 신선도 보장 정책(파손·부패 A/S 프로세스, 도착 당일 신고 시 신속보상).
  - 5스크린: 보관/세척/숙성 팁, 취식 시기 가이드(고객 불만 선제 차단).
- 전환 요소
  - 리뷰 앵커 모듈(‘당도계 사진 리뷰’ 모음), N페이 적립/쿠폰, 배송마감 카운트다운, 당일선별-출고타임라인.
  - 네이버 속성 태깅(품종/원산지/중량/보관/배송유형) 100% 완료.

## 8. 네이버 알고리즘 관점 추정
- 데이터 기반 관찰
  - 제목 키워드 매칭, 옵션 범위(모음전), 썸네일 CTR 유도형이 상위권 다수.
- 전략적 추정
  - 랭킹 신호: 키워드 적합성 + CTR + 전환/이탈 + 리뷰품질(포토비중/최신성) + 스토어지수(발송지연/CS) + 가격경쟁력.
  - 보기(View)/블로그 링크성 신호가 브랜드·스토어 신뢰에 간접 기여.
  - 동일 키워드 다발은 단기 노출엔 유리하나, 중장기엔 사용자 반응(CTR/체류/전환)이 핵심. 그러므로 “과밀 키워드” 대신 “핵심 2~3키워드 + 품종/용량 명확화”가 효율적.

- 촬영

- 운영
  - 당일 수확·선별·출고 타임라인 기록, 택배 마감 시각 공지, 주말·공휴일 출고 정책, 클레임 체크리스트(사진 2장·박스 라벨·온도).

## 11. 상세페이지 수정 방향
- 제목
  - 현재: 키워드 나열 과다, 의사결정 정보 분산
  - 개선 템플릿 예시
    - 씨없는 고당도 블랙사파이어 2kg | 18~21brix | 당일선별 냉장출고
    - 20brix 애플청포도 1kg | 아삭달콤 | 주말 제외 당일발송
- 썸네일
  - 1:1 비율, 과실 클로즈업 + ‘씨없는/18~21brix’ 소형 배지 2개 이내, 과도한 텍스트 금지, 실제 포장컷 1장 추가(신뢰).
- 퍼스트뷰(상단 3스크롤)
  - 1) 가치 제안: 씨없는·브릭스·식감(크런치) 한 눈에
  - 2) 산지 신뢰: 로트 브릭스 실측/수확일/선별기준 카드
  - 3) 옵션 선택: 500g 체험→1kg/2kg 패밀리→3~4kg 카페, 예상 인원/소비일수 안내
- 신뢰/정책
  - 실중량 보장 범위(±값), 도착 당일 신고 시 신속보상, 여름철 ‘제빙팩 응결수’ 안내, 알 터짐 허용 범위 명시
- 정보성 모듈
  - 보관/세척/취식 팁, 품종 비교 차트(블랙사파이어/크림슨/어텀크리스피), 당도·산미 레이더차트
- 리뷰 큐레이션
  - ‘브릭스계 사진 리뷰’ 모아보기, ‘카페 대용량 사용 후기’ 섹션

## 12. 결론
- 데이터 기반 결론
  - 네이버 포도 키워드 시장은 ‘씨없는/고당도/브릭스’로 클릭을 만들고, 품종명 다발·혼합옵션으로 롱테일 검색을 흡수한다. 대용량 수요와 프리미엄 수요가 공존.
- 전략적 권고
  - ONEFLOW는 네이버 특유의 검색-뷰/블로그-리뷰-스토어 신뢰의 연쇄를 설계해야 한다. 상세페이지는 키워드 과밀을 줄이고 ‘브릭스·산지·콜드체인’의 근거를 시각화하라. 블로그/VIEW에서 정보·현장형 콘텐츠로 신뢰를 축적하고, 리뷰를 구조화해 전환을 증폭하라. 이를 로트 데이터와 운영 SOP로 자동화하면 산지직송 플랫폼의 차별화가 완성된다.
